Keyboard Accessory
======================
ViewController to add additional bars on top of the iOS 7.1 keyboard. Apple's documentation can be viewed, <a href="https://developer.apple.com/library/ios/documentation/ToolsLanguages/Reference/UIAKeyboardClassReference/UIAKeyboard/UIAKeyboard.html#//apple_ref/doc/uid/TP40009904">here</a>, and a video demonstration of the code, <a href="https://www.dropbox.com/s/cc8frjf39e3oso4/KeyboardAccessory.mov">here</a>.

Requirements

- iOS 7.1 non-ipad runtime.
- Xcode Project.
